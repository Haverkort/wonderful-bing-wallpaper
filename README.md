### wonderful-bing-wallpaper

🌅A simple bing daily wallpaper api lib. 极致简洁的 Node.js 版必应壁纸 API 库。

- 简单：1个接口
- 稳定：Bing 不挂库不挂
- 轻巧：不依赖任何第三方

### 贡献者

[stackoverflow - Is there a way to get Bing's photo of the day?](https://stackoverflow.com/a/18096210/6222535) 

### 怎么用

```bash
npm i wonderful-bing-wallpaper --save
```

```javascript
const wbda = require('wonderful-bing-wallpaper')

// wbda()
```

### 测试执行

```
npm run test
npm run example
```
